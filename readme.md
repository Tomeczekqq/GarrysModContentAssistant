# Garry's Mod Content Assistant v0.3 
Easy Garry's Mod Content installation tool. You can install gmod content in just 2 clicks.
Download compiled (**.exe**) on our [website](https://gamebanana.com/tools/6292)

## Requirements
- Python 3
- pySmartDL
- bs4
- requests
